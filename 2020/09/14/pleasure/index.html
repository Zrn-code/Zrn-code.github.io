# 關於此網站

# 簡介

這個網站是我的第一重要的主網站，目前的目標就是更新之前寫的 code 和精彩的教學部分。

# 目標

  • 希望我的學習歷程可以寫得越來越好，超越許多不凡的大神啊～～希望有動力一直持續下去。
  • 希望可以透過這個網站遇到更多大神指點。
  • 希望可以再多寫一些比較厲害的內容。
  • 衝刺吧！！！ 人工智慧 、 資料結構 以及 前後端建設 。

# 未來規劃

預計，再過個一、兩年，等到我高中畢業以後，我的主要網站就會只剩一個。
也就是，現在的這個網站，會取代掉其他的兩個主要網站，這樣就可以減少一些連結問題了，雖然，這網站可能會有點亂啦。
