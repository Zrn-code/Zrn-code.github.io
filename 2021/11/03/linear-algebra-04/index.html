# 線性變換 linear transformation

# 定義

令V,WV,WV,W 都是建立在體FFF 的向量空間， T:V→WT:V \rightarrow WT:V→W 稱作VVV 的WWW 的線性轉換。
並且，TTT 也滿足以下兩點:

  1. T(x+y)=T(x)+T(y)∀x,y∈VT(x+y) = T(x) + T(y) \forall x,y \in VT(x+y)=T(x)+T(y)∀x,y∈V
  2. T(cx)=cT(x)∀c∈F,∀x∈VT(cx) = cT(x) \forall c \in F ,\forall x \in VT(cx)=cT(x)∀c∈F,∀x∈V
小筆記
  1. TTT 是線性函數
  2. 在定義下，T(0)=0,T(cx+y)=cT(x)+T(y)T(0) = 0, T(cx+y) = cT(x)+T(y)T(0)=0,T(cx+y)=cT(x)+T(y)

# 範例們

# 範例一

  1. x↦0x \mapsto 0x↦0 是一個線性變換，稱作為 0 變換。
  2. x↦xx \mapsto xx↦x 是一個線性變換，稱作為單位變換。
小筆記

所有由V→WV \to WV→W 的線性變換會形成向量空間，記作 L(V,W)L(V,W)L(V,W)。
若此時的V=WV = WV=W，則記作 L(V)L(V)L(V)。

# 範例二

  1. x↦mxx \mapsto mxx↦mx 是線性變換。
  2. x↦x+2x \mapsto x+2x↦x+2 不是線性變換。 0↦2≠00 \mapsto 2 \neq 00↦2​=0
  3. x↦x2x \mapsto x^2x↦x2 不是線性變換。 T(1)+T(2)=5≠T(3)=9T(1)+T(2) = 5 \neq T(3) = 9T(1)+T(2)=5​=T(3)=9

# 範例三

  1. (a,b)↦(a,−b)(a,b) \mapsto (a,-b)(a,b)↦(a,−b) 是線性變換。
  2. (a,b)↦(a,0)(a,b) \mapsto (a,0)(a,b)↦(a,0) 是線性變換。
  3. (a,b)↦(pa,qb)(a,b) \mapsto (pa,qb)(a,b)↦(pa,qb) 是線性變換。
  4. (a,b)↦(acosθ−bsinθ,asinθ+bcosθ)(a,b) \mapsto (acos\theta -bsin\theta, asin\theta+bcos\theta)(a,b)↦(acosθ−bsinθ,asinθ+bcosθ) 是線性變換。(旋轉矩陣)
  5. (a,b)↦(2a+b,a2)(a,b) \mapsto (2a+b,a^2)(a,b)↦(2a+b,a2) 不是線性變換。

# 範例四

  1. A↦ATA \mapsto A^TA↦AT 是線性變換。
  2. A↦det(A)A \mapsto det(A)A↦det(A) 不是線性變換。
  3. A↦tr(A)A \mapsto tr(A)A↦tr(A) 是線性變換。
    tr(A)=∑i=1naiitr(A)=\sum_{i=1}^na_{ii}tr(A)=∑i=1n​aii​ 稱作 A 矩陣的跡。

# 核 kernel & 零化度 nullity

# 定義

設 VVV 和 WWW 是向量空間，設 TTT 是從 VVV 到 WWW 的線性變換。
kerT:={v∈V:Tv=0w}\mathop{\mathrm{ker}}\, T := \{\mathbf{v} \in V : T\mathbf{v} = \mathbf{0}_{w}\}kerT:={v∈V:Tv=0w​}
kerT=N(T)kerT = N(T)kerT=N(T) 稱作TTT 的核或零空間。

# 範例們

  1. x↦0x \mapsto 0x↦0 => kerT=VkerT =VkerT=V
  2. x↦xx \mapsto xx↦x => kerT=0kerT = 0kerT=0
  3. (x,y)↦x−y(x,y) \mapsto x-y(x,y)↦x−y => kerT={x−y=0∣x,y∈R}kerT =\{ x-y=0|x,y \in \mathbb{R} \}kerT={x−y=0∣x,y∈R}
  4. (x,y)↦(x+y,0,2x−y)(x,y) \mapsto (x+y,0,2x-y)(x,y)↦(x+y,0,2x−y) => kerT=(0,0)kerT =(0,0)kerT=(0,0)
  5. f(x)↦f′(x)f(x) \mapsto f'(x)f(x)↦f′(x) => kerT={c∈R}kerT = \{ c \in \mathbb{R} \}kerT={c∈R}
  6. (x,y)↦(x,0)(x,y) \mapsto (x,0)(x,y)↦(x,0) => kerT={(0,y)∣y∈R}kerT = \{ (0,y) | y \in \mathbb{R} \}kerT={(0,y)∣y∈R}

# 定理

若T:V→WT:V \rightarrow WT:V→W 是一個線性變換。
kerTkerTkerT 會是 VVV 的子空間。
可定義 dim(kerT):=n(T)dim(kerT) := n(T)dim(kerT):=n(T) 稱為TTT 的零化度

# 像 image & 秩 rank

# 定義

設 VVV 和 WWW 是向量空間，設 TTT 是從 VVV 到 WWW 的線性變換。
imT:={Tv∣v∈V}\mathop{\mathrm{im}}\, T := \{ T\mathbf{v} | \mathbf{v} \in V \}imT:={Tv∣v∈V}
imT=R(T)imT = R(T)imT=R(T) 稱作TTT 的像或值域。

# 範例們

  1. x↦0x \mapsto 0x↦0 => kerT=0kerT = 0kerT=0
  2. x↦xx \mapsto xx↦x => kerT=VkerT =VkerT=V
  3. (x,y)↦x−y(x,y) \mapsto x-y(x,y)↦x−y => imT=RimT = RimT=R
  4. (x,y)↦(x+y,0,2x−y)(x,y) \mapsto (x+y,0,2x-y)(x,y)↦(x+y,0,2x−y) => imT={(a,0,b)∣a,b∈R}imT =\{(a,0,b) | a,b \in \mathbb{R} \}imT={(a,0,b)∣a,b∈R}
  5. f(x)↦f′(x)f(x) \mapsto f'(x)f(x)↦f′(x) => imT=Pn−1(R)imT = P_{n-1}(R)imT=Pn−1​(R)
  6. (x,y)↦(x,0)(x,y) \mapsto (x,0)(x,y)↦(x,0) => imT={(x,0)∣x∈R}imT = \{ (x,0) | x \in \mathbb{R} \}imT={(x,0)∣x∈R}

# 定理 1

imTimTimT 會是 WWW 的子空間。

=> T(β)={T(v1),T(v2),...,T(vn)}T(\beta) = \{ T(v_1),T(v_2),...,T(v_n) \}T(β)={T(v1​),T(v2​),...,T(vn​)} 會是 imT 的基底。

=> n(T)+r(T)=dimVn(T)+r(T) = dimVn(T)+r(T)=dimV
