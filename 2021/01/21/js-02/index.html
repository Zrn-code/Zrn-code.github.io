# 創建 JS 檔案

Windows 預設將副檔名隱藏，因此，我們可以進入檢視的設定，將副檔名的選項點開。

接著選擇一個資料夾來放置 JS 原始碼檔案 (教學部分皆會將檔案放置於 C:\Users\user\Desktop\js )
