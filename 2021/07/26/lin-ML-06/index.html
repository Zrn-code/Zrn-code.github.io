上一節課，我們主要探討了當 M 的數值大小對機器學習的影響。如果 M 很大，那麼就不能保證機器學習有很好的泛化能力，所以問題轉換為驗證 M 有限，即最好是按照多項式成長。然後通過引入了成長函數mH(N)m_H(N)mH​(N) 和 dichotomy 以及 break point 的概念，提出 2D perceptrons 的成長函數mH(N)m_H(N)mH​(N) 是多項式級別的猜想。這就是本節課將要深入探討和證明的內容。

# Restriction of Break Point

我們先回顧一下上節課的內容，四種成長函數與 break point 的關係：

下面引入一個例子，如果 k=2，那麼當 N 取不同值的時候，計算其成長函數mH(N)m_H(N)mH​(N) 是多少。很明顯，當 N=1 時，mH(N)m_H(N)mH​(N)；當 N=2 時，由 break point 為 2 可知，任意兩點都不能被 shattered（shatter 的意思是對 N 個點，能夠分解為2N2^N2N 種 dichotomies）；mH(N)m_H(N)mH​(N) 最大值只能是 3；當 N=3 時，簡單繪圖分析可得其mH(N)=4m_H(N)=4mH​(N)=4，即最多只有 4 種 dichotomies。

所以，我們發現當N>kN>kN>k 時，break point 限制了mH(N)m_H(N)mH​(N) 值的大小，也就是說影響成長函數mH(N)m_H(N)mH​(N) 的因素主要有兩個：

  • 抽樣資料集 N
  • break point k（這個變數確定了假設的類型）
    那麼，如果給定 N 和 k，能夠證明其mH(N)m_H(N)mH​(N) 的最大值的上界是多項式的，則根據霍夫丁不等式，就能用mH(N)m_H(N)mH​(N) 代替 M，得到機器學習是可行的。所以，證明mH(N)m_H(N)mH​(N) 的上界是 poly (N)，是我們的目標。

# Bounding Function: Basic Cases

求解 B (N,k) 的過程十分巧妙：

  • 當 k = 1 時，B(N,1)B(N,1)B(N,1) 恆為 1。
  • 當 N < k 時，根據 break point 的定義，很容易得到B(N,k)=2NB(N,k)=2^NB(N,k)=2N。
  • 當 N = k 時，此時 N 是第一次出現不能被 shatter 的值，所以最多只能有2N−12^N-12N−1 個 dichotomies，則B(N,k)=2N−1B(N,k)=2^N-1B(N,k)=2N−1。

以 B (4,3) 為例，首先想著能否構建 B (4,3) 與 B (3,x) 之間的關係。
首先，把 B (4,3) 所有情況寫下來，共有 11 組。也就是說再加一種 dichotomy，任意三點都能被 shattered，11 是極限。

對這 11 種 dichotomy 分組，目前分成兩組，分別是 orange 和 purple，orange 的特點是，x1,x2 和 x3 是一致的，x4 不同並成對，例如 1 和 5，2 和 8 等，purple 則是單一的，x1,x2,x3 都不同，如 6,7,9 三組。

將 Orange 去掉 x4 後去重得到 4 個不同的 vector 並成為α\alphaα，相應的 purple 為β\betaβ。那麼 B (4,3) = 2α+β2\alpha + \beta2α+β，這個是直接轉化。緊接著，由定義，B (4,3) 是不能允許任意三點 shatter 的，所以由α\alphaα 和β\betaβ 構成的所有三點組合也不能 shatter（alpha 經過去重），即α+β≤B(3,3)\alpha + \beta\leq B(3,3)α+β≤B(3,3)。

# 總結

本節課我們主要介紹了只要存在 break point，那麼其成長函數mH(N)m_H(N)mH​(N) 就滿足 poly (N)。推導過程是先引入mH(N)m_H(N)mH​(N) 的上界 B (N,k)，B (N,k) 的上界是 N 的 k-1 階多項式，從而得到mH(N)m_H(N)mH​(N) 的上界就是 N 的 k-1 階多項式。然後，我們通過簡單的三步證明，將mH(N)m_H(N)mH​(N) 代入了 Hoffding 不等式中，推導出了 Vapnik-Chervonenkis (VC) bound，最終證明了只要 break point 存在，那麼機器學習就是可行的。
