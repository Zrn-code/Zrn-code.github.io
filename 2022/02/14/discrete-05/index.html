# 一般生成函數

# 定義

設一數列為:a0,a1,a2...,ana_0,a_1,a_2...,a_na0​,a1​,a2​...,an​
定義A(x)=a0+a1x+...+anxn+...=∑n=0∞anxnA(x) = a_0+a_1x+...+a_nx^n+... = \sum^{\infty}_{n=0}a_nx^nA(x)=a0​+a1​x+...+an​xn+...=∑n=0∞​an​xn
稱為ana_nan​ 的生成函數 generating function(GF)

# 公式

  1. (1+x)n=∑r=0n(nr)xr(1+x)^n= \sum^n_{r=0}\binom{n}{r}x^r(1+x)n=∑r=0n​(rn​)xr: 二項式定理
  2. 1+x+x2+...=11−x1+x+x^2+...=\frac{1}{1-x}1+x+x2+...=1−x1​: 無窮等比級數
  3. 1+x+x2+...+xn=11−x−xn+11−x=1−xn+11−x1+x+x^2+...+x^n=\frac{1}{1-x}-\frac{x^{n+1}}{1-x} = \frac{1-x^{n+1}}{1-x}1+x+x2+...+xn=1−x1​−1−xxn+1​=1−x1−xn+1​

# 例子

  1. (n0)+(n1)+...+(nn)=2n\binom{n}{0}+\binom{n}{1}+...+\binom{n}{n} = 2^n(0n​)+(1n​)+...+(nn​)=2n
  2. 20(n0)+21(n1)+...+2n(nn)=3n2^0\binom{n}{0}+2^1\binom{n}{1}+...+2^n\binom{n}{n} = 3^n20(0n​)+21(1n​)+...+2n(nn​)=3n
  3. 1(n0)+2(n1)+...+n(nn)=n2n−11\binom{n}{0}+2\binom{n}{1}+...+n\binom{n}{n} = n2^{n-1}1(0n​)+2(1n​)+...+n(nn​)=n2n−1
  4. ∑r=0n(nr)xr=(1+x)n\sum^n_{r=0}\binom{n}{r}x^r = (1+x)^n∑r=0n​(rn​)xr=(1+x)n
note

(nr)=n!r!(n−r)!=n(n−1)(n−2)...(n−r+1)r!,n∈R\binom{n}{r} = \frac{n!}{r!(n-r)!} = \frac{n(n-1)(n-2)...(n-r+1)}{r!},n\in \mathbb{R}(rn​)=r!(n−r)!n!​=r!n(n−1)(n−2)...(n−r+1)​,n∈R: 代數定義
