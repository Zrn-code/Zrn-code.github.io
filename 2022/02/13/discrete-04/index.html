# 基本計數原理

  • 乘法原理: 獨立且連續事件: m×nm \times nm×n
  • 加法原理: 互斥事件: m+nm+nm+n

# 排列

# 常見公式

  1. nnn 件相異物，取rrr 件物品排列 (不重複排列)⇒n!(n−r)!=Prn\Rightarrow \frac{n!}{(n-r)!} = P^n_r⇒(n−r)!n!​=Prn​
  2. nnn 件相異物，取rrr 件物品排列 (允許重複排列)⇒nr\Rightarrow n^r⇒nr
  3. nnn 件相異物，全排列⇒n!\Rightarrow n!⇒n!
  4. nnn 件相異物，環狀排列⇒n!n=(n−1)!\Rightarrow \frac{n!}{n} = (n-1)!⇒nn!​=(n−1)!
  5. nnn 件不全相異的物品，共有rrr 種，第一種有n1n_1n1​ 個，第二種有n2n_2n2​ 個，...，第rrr 種有nrn_rnr​ 個，
    全排列⇒n!n1!n2!...nr!\Rightarrow \frac{n!}{n_1!n_2!...n_r!}⇒n1​!n2​!...nr​!n!​

# 組合

# 常見公式

  1. nnn 件相異物，取rrr 件物品組合 (不允許重複)⇒Prnr!=n!(n−r)!r!=Crn=(nr)\Rightarrow \frac{P^n_r}{r!} = \frac{n!}{(n-r)!r!} = C^n_r = \binom{n}{r}⇒r!Prn​​=(n−r)!r!n!​=Crn​=(rn​)
  2. nnn 件相異物，取rrr 件物品組合 (允許重複)⇒(n+r−1r)=Hrn\Rightarrow \binom{n+r-1}{r} = H^n_r⇒(rn+r−1​)=Hrn​
  3. x1+x2+...+xn=rx_1 + x_2 +...+x_n=rx1​+x2​+...+xn​=r 之正整數解⇒(r−1n−1)\Rightarrow \binom{r-1}{n-1}⇒(n−1r−1​)
