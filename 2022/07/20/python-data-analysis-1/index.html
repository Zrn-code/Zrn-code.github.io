# Intro

Pandas 結合 NumPy 的特性，以及試算表和關連式資料庫（SQL）的資料操作能力，可用來對資料進行重構、切割、聚合等操作。使你能夠快速的發現資料中的資訊以及其中蘊藏的意義。
