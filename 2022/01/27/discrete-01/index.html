集合 set 為一堆物品的收集

  1. x∈A↔xx \in A \leftrightarrow xx∈A↔x 為AAA 的元素

  3. A⊆B↔(∀x∈A→x∈B)A \subseteq B \leftrightarrow ( \forall x \in A \rightarrow x \in B)A⊆B↔(∀x∈A→x∈B)
  4. A⊂B↔(A⊆B∧A≠B)A \subset B \leftrightarrow (A \subseteq B \wedge A \neq B)A⊂B↔(A⊆B∧A​=B)
  5. A=B↔(A⊆B∧B⊆A)A=B \leftrightarrow (A \subseteq B \wedge B \subseteq A)A=B↔(A⊆B∧B⊆A)

  1. N={0,1,2,3,...}\mathbb{N} = \{ 0, 1, 2, 3,...\}N={0,1,2,3,...} 自然數
  2. Z={...,−2,−1,0,1,2,...}\mathbb{Z} = \{ ...,-2,-1,0,1,2,...\}Z={...,−2,−1,0,1,2,...} 整數
  3. Z+={1,2,3,4...}\mathbb{Z}^{+} = \{ 1,2,3,4... \}Z+={1,2,3,4...} 正整數

# 運算

  1. 聯集 union : A∪B={x∣x∈A∪x∈B}.A \cup B = \{ x | x \in A \cup x \in B \}.A∪B={x∣x∈A∪x∈B}.
  2. 交集 intersection : A∩B={x∣x∈A∩x∈B}.A \cap B = \{ x | x \in A \cap x \in B \}.A∩B={x∣x∈A∩x∈B}.
  3. 差集 difference : A−B={x∣x∈A∩x∉B}.A-B = \{ x| x \in A \cap x \notin B \}.A−B={x∣x∈A∩x∈/​B}.
  4. 補集 complement : A‾={x∣x∉A}.\overline{A} = \{ x | x \notin A \}.A={x∣x∈/​A}.
  5. 對稱差 symmetric difference : A⊕B=(A∪B)−(A∩B)A\oplus B = (A \cup B) - (A \cap B)A⊕B=(A∪B)−(A∩B)

# 交換性

(A∪B)∪C=A∪(B∪C)(A\cup B) \cup C = A \cup ( B \cup C)(A∪B)∪C=A∪(B∪C)
(A∩B)∩C=A∩(B∩C)(A\cap B) \cap C = A \cap ( B \cap C)(A∩B)∩C=A∩(B∩C)

# 分配性 (不同運算)

A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C)A∪(B∩C)=(A∪B)∩(A∪C)
A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C)A∩(B∪C)=(A∩B)∪(A∩C)
A∩(B⊕C)=(A∩B)⊕(A∩C)A \cap (B \oplus C) = (A \cap B) \oplus (A \cap C)A∩(B⊕C)=(A∩B)⊕(A∩C) (交集有分配性)
A∪(B⊕C)=(A∪B)⊕(A∪C)A \cup (B \oplus C) = (A \cup B) \oplus (A \cup C)A∪(B⊕C)=(A∪B)⊕(A∪C) (聯集無分配性)
