# 函數

f⊆A×Bf \subseteq A \times Bf⊆A×B 滿足:∀a∈A∃!b∈B:afb\forall a \in A \exists !b \in B : afb∀a∈A∃!b∈B:afb
稱fff 為AAA 至BBB 的函數 function relation ，記作f:A→Bf: A \rightarrow Bf:A→B

# 定義

設fff 為A→BA \rightarrow BA→B 函數

  1. AAA 稱作定義域 domain
  2. BBB 稱作對應域 codomain
  3. A1⊆A,f(A1)={f(a)∣a∈A1}A_1 \subseteq A, f(A_1) = \{ f(a) | a \in A_1 \}A1​⊆A,f(A1​)={f(a)∣a∈A1​}
  4. B1⊆B,f−1(B1)={f(a)∣a∈B1}B_1 \subseteq B, f^{-1}(B_1) = \{ f(a) | a \in B_1 \}B1​⊆B,f−1(B1​)={f(a)∣a∈B1​}
  5. f(A)f(A)f(A) 稱作值域，即被AAA 對應到的部分
note

f−1f^{-1}f−1 代表的意涵為反向，不一定可逆
如果可逆的話，f−1f^{-1}f−1 即可稱為反函數。

# 定義

設fff 為A→BA \rightarrow BA→B 函數

  1. 若a1≠a2→f(a1)≠f(a2)a_1 \neq a_2 \rightarrow f(a_1) \neq f(a_2)a1​​=a2​→f(a1​)​=f(a2​)，也就是說，f(a1)=f(a2)→a1=a2f(a_1) = f(a_2) \rightarrow a_1 = a_2f(a1​)=f(a2​)→a1​=a2​
    稱fff 為一對一函數 one-to-one 、 injective
  2. 若f(A)=Bf(A) = Bf(A)=B，稱fff 為映射函數 onto 、 subjective
  3. 若fff 為 1-1 且 onto ，則稱fff 為可逆函數 invertible 、 bijection

# 定理

設fff 為A→BA \rightarrow BA→B 函數，且A1,A2∈A,B1,B2∈BA_1,A_2 \in A, B_1,B_2 \in BA1​,A2​∈A,B1​,B2​∈B

  1. A1⊆A2→f(A1)⊆f(A2)A_1 \subseteq A_2 \rightarrow f(A_1) \subseteq f(A_2)A1​⊆A2​→f(A1​)⊆f(A2​)
  2. f(A1∪A2)=f(A1)∪f(A2)f(A_1 \cup A_2) = f(A_1) \cup f(A_2)f(A1​∪A2​)=f(A1​)∪f(A2​)
  3. f(A1∩A2)⊆f(A1)∩f(A2)f(A_1 \cap A_2) \subseteq f(A_1) \cap f(A_2)f(A1​∩A2​)⊆f(A1​)∩f(A2​)，如果fff 是多對一
  4. f(A1∩A2)=f(A1)∩f(A2)f(A_1 \cap A_2) = f(A_1) \cap f(A_2)f(A1​∩A2​)=f(A1​)∩f(A2​)，如果fff 是一對一
  5. B1⊆B2→f−1(B1)⊆f−1(B2)B_1 \subseteq B_2 \rightarrow f^{-1}(B_1) \subseteq f^{-1}(B_2)B1​⊆B2​→f−1(B1​)⊆f−1(B2​)
  6. f−1(B1∪B2)=f−1(B1)∪f−1(B2)f^{-1}(B_1 \cup B_2) = f^{-1}(B_1) \cup f^{-1}(B_2)f−1(B1​∪B2​)=f−1(B1​)∪f−1(B2​)
  7. f−1(B1∩B2)=f−1(B1)∩f−1(B2)f^{-1}(B_1 \cap B_2) = f^{-1}(B_1) \cap f^{-1}(B_2)f−1(B1​∩B2​)=f−1(B1​)∩f−1(B2​)
note

設fff 為A→BA \rightarrow BA→B 函數
fff 為 1-1 且 onto ，fff 的反關係為函數，稱為 inverse function ，記作f^

# 合成

設fff 為A→BA \rightarrow BA→B 函數，ggg 為B→CB \rightarrow CB→C 函數
g∘f:A→C⇔(g∘f)(a)=g(f(a))g \circ f: A \rightarrow C \Leftrightarrow (g \circ f)(a) = g(f(a))g∘f:A→C⇔(g∘f)(a)=g(f(a))(先作fff)

# 定理

設fff 為A→BA \rightarrow BA→B 函數，ggg 為B→CB \rightarrow CB→C 函數

  1. f 為 1-1 且 g 為 1-1 →g∘f\rightarrow g \circ f→g∘f 為 1-1
