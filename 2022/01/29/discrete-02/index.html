# 反身性 Reflexive / 非反身性 Irreflexive

# 反身性 <-> 不具反身性

RRR 具有反身性↔∀a∈A,aRa\leftrightarrow \forall a \in A, aRa↔∀a∈A,aRa 有 (a,a)
反之，則稱為不具反身性 (對角線不全為 1)

eg.A={1,2,3},R={(1,1),(1,2),(2,3),(2,2),(3,3)}A = \{ 1,2,3 \},R = \{ (1,1),(1,2),(2,3),(2,2),(3,3) \}A={1,2,3},R={(1,1),(1,2),(2,3),(2,2),(3,3)}
→\rightarrow→ 每一個都和自己有關係，RRR 就具反身性

# 非反身性 <-> 不具非反身性

RRR 具有非反身性↔∀a∈A,aR‾a\leftrightarrow \forall a \in A, a\overline{R}a↔∀a∈A,aRa 無 (a,a)
反之，則稱為不具反身性 (對角線不全為 0)

eg.A={1,2,3},R={(1,2),(2,3),(1,3)}A = \{ 1,2,3 \},R = \{ (1,2),(2,3),(1,3) \}A={1,2,3},R={(1,2),(2,3),(1,3)}
→\rightarrow→ 每一個都和自己沒關係，RRR 就具非反身性

note

非反身性 不是 不具反身性
但是，反身性一定不具非反身性

note

若∣A∣=n|A| = n∣A∣=n，則:

  1. AAA 上的 二元關係 個數=2n2= 2^{n^{2}}=2n2(1 或 0)
  2. AAA 上的 反身性關係 個數=2n2−n= 2^{n^{2}-n}=2n2−n(對角線全為 1)
  3. AAA 上的 非反身性關係 個數=2n2−n= 2^{n^{2}-n}=2n2−n(對角線全為 0)
