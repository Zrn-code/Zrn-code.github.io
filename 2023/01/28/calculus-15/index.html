# Implicit Differentiation

⇒f(x,y,g(x,y))=0⇒fx+fydydx+fz∂z∂x=0\Rightarrow f(x,y,g(x,y)) = 0 \Rightarrow f_x + f_y\frac{dy}{dx} + f_z\frac{\partial z}{\partial x}=0⇒f(x,y,g(x,y))=0⇒fx​+fy​dxdy​+fz​∂x∂z​=0
Since dydx=0⇒∂z∂x=−fxfz\frac{dy}{dx} = 0 \Rightarrow \frac{\partial z}{\partial x} = -\frac{f_x}{f_z}dxdy​=0⇒∂x∂z​=−fz​fx​​.

# Maximum and Minimum Values

fff is continuous on a closed and bounded set DDD.

  1. There exist absolute minimum and absolute maximum.
  2. Those extreme points occur at the critical points of fff or the boundary of DDD.

# Lagrange Multipliers

  • 想法:
  1. 極值產生的地方:
    f 的 level curve or level surface 和 ggg 的 level curve or level surface 相切。
  2. Level curve of f \prep \nabla f
  • 結論: Extrma occur when
    1. ∇f=λ∇g\nabla f = \lambda \nabla g∇f=λ∇g for some λ∈R\lambda \in \mathbb{R}λ∈R.
    2. ∇f=λ∇g+μ∇h\nabla f = \lambda \nabla g + \mu \nabla h∇f=λ∇g+μ∇h for some λ,μ∈R\lambda,\mu \in \mathbb{R}λ,μ∈R.
